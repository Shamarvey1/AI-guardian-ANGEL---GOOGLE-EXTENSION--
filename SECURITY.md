# 🔐 Security Policy

## 📌 Supported Versions

We currently support only the latest version of this project.

| Version | Supported |
|---------|----------|
| Latest  | ✅ Yes   |
| Older   | ❌ No    |

---

## 🚨 Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

### 📧 How to report:

- Do NOT open a public GitHub issue for security vulnerabilities
- Contact maintainers directly via GitHub

---

## 📝 What to include:

- A clear description of the vulnerability  
- Steps to reproduce  
- Possible impact  
- Screenshots/logs (if available)  

---

## ⏱️ Response Time

We aim to respond within **48–72 hours**.

---

## 🛡️ Responsible Disclosure

We take security issues seriously and will investigate them as soon as possible.

Please allow time for fixing issues before public disclosure.

Thank you for helping keep this project secure!
